Init libScroll library
(the best way to do it, call libScrollInit command in openStack script).

Connect scroller to scrolling groups. 

To do it you need to call libScrollConnect command with the long id of group to scroll and direction(E.g. libScrollConnect the long id of group "toScroll", "vertical"), if pDirection is empty then the scroller will work on both directions. 

Note: Call libScrollConnect command from card where the scrolling group are.

Enable scroll using libScrollEnable command with the long id of group to scroll.

Disable scroll using libScrollDisable command.

command libScrollDelete  for deleting the scroller.
